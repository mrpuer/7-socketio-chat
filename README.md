# Socket.io realtime chat

Home work for the Netology.ru lesson _"Real-time interaction with Socket.io"_.

Using:
* `npm start`
* Go to [http://localhost:3000/](http://localhost:3000/)

Used tools:
+ NodeJS
+ Express
+ Socket.io
+ JQuery

Features:
*   sending messages
*   receiving messages
*   notification of users about adding a new user.
